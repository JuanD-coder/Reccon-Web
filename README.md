# RECCON - Recolección y Contabilidad

![RecCon Hero](/public/images/screenshots/hero.png)

## ☕ ¿Qué es RECCON?

**RECCON** es una solución tecnológica avanzada diseñada específicamente para el sector caficultor. Nuestra misión es empoderar al campo colombiano mediante la digitalización de procesos que tradicionalmente se han llevado en papel, permitiendo que cada finca funcione con la precisión de una empresa moderna.

Con RECCON, los caficultores pueden llevar un control riguroso de sus recolecciones, gastos y personal, todo desde la comodidad de su dispositivo móvil, permitiendo una visión clara de la rentabilidad de su cosecha.

---

## 🚀 Características Principales

- **100% Offline**: El campo no siempre tiene señal, pero RECCON no la necesita para funcionar. Registra todo en tiempo real y sincroniza cuando vuelvas a tener conexión.
- **Registros Digitales Rápidos**: Olvida los cuadernos perdidos y la letra ilegible. Anota recolecciones en segundos.
- **Contabilidad de Gastos**: Mantén tus cuentas claras. El sistema suma kilos y calcula liquidaciones automáticamente, eliminando errores humanos.
- **Reportes Profesionales**: Exporta tus bitácoras a Excel para trámites legales o análisis detallados de productividad.
